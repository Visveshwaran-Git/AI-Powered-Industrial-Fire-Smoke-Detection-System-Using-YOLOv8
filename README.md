# AI-Powered Industrial Fire & Smoke Detection System Using YOLOv8

## Overview

The AI-Powered Industrial Fire & Smoke Detection System is a real-time safety monitoring solution designed to detect fire and smoke incidents in industrial environments using Computer Vision and Deep Learning. The system leverages the YOLOv8 object detection model to identify hazardous events from live camera feeds and video streams, enabling early warning and rapid response to potential fire emergencies.

In addition to fire and smoke detection, the system can monitor people and vehicles present in the scene, providing enhanced situational awareness and intelligent risk assessment. The solution is designed to improve workplace safety, reduce response times, and minimize damage caused by industrial accidents.

## Features

* Real-time fire detection using YOLOv8
* Real-time smoke detection using Deep Learning
* Detection of people and vehicles in hazardous zones
* Live webcam and video file support
* Intelligent risk level classification (Clear, Caution, Warning, Critical)
* Real-time visual alerts and monitoring dashboard
* Bounding box visualization with confidence scores
* High-performance inference using PyTorch and YOLOv8
* Industrial safety monitoring and surveillance applications

## Technologies Used

* Python
* YOLOv8
* PyTorch
* OpenCV
* NumPy
* Computer Vision
* Deep Learning

## System Workflow

1. Capture video frames from a camera or video source.
2. Process frames using the YOLOv8 object detection model.
3. Detect fire, smoke, people, and vehicles.
4. Calculate confidence scores for detected hazards.
5. Assess overall risk levels based on detected threats.
6. Display real-time alerts and visual indicators.
7. Generate output video with detection annotations.

## Applications

* Industrial Safety Monitoring
* Manufacturing Plants
* Warehouses
* Chemical Industries
* Oil and Gas Facilities
* Smart Surveillance Systems
* Fire Prevention and Emergency Response

## Project Objective

The primary objective of this project is to develop an intelligent AI-based surveillance system capable of detecting fire and smoke incidents at an early stage, helping industries improve safety standards, prevent accidents, and protect human lives and infrastructure.
